# GIS Capstone: Cattle Habitat Preference Modeling

## Project overview
In this capstone project, I analyzed GPS collar data and environmental covariates to model cattle habitat preferences across seasonal windows. The objective was to identify which landscape factors influence habitat use and to produce maps that can support rangeland planning decisions.

## Methods
- Cleaned and standardized location records from multiple GPS devices.
- Derived terrain and vegetation predictors from raster datasets.
- Built exploratory models in R to test predictor importance.
- Created cartographic outputs in ArcGIS Pro and Adobe Illustrator.

## Key outputs
- Habitat suitability map atlas by season.
- Short technical methods note for non-technical stakeholders.
- Validation summary showing model strengths and limitations.

## Reflection
This project strengthened my ability to connect reproducible analysis with communication-ready outputs. A major lesson was balancing model complexity with interpretability for decision-makers.
